# Crux -- Local Synthetic Data Engine

### **Tech Stack**
`FastAPI` • `PySide6` • `CTGAN` • `Ollama` • `Pandas` • `Python`

---

### **Key Features & Achievements**
*   **Local-First CSV Operations**: Built a privacy-focused CSV cleaning, data repair, and synthetic data generation tool featuring both separate web-based and standalone desktop client packages.
*   **Deep Integrations**: Combined `FastAPI` backends with `PySide6` GUIs, `CTGAN` deep generative models, `Pandas` computation engines, and local **Ollama (Llama 3)** workflows.
*   **Query-Aware AI Chatbot**: Built an integrated LLM chat interface capable of executing structured queries to synthesize and alter datasets.
*   **Recruiter & Developer Highlights**:
    *   Side-by-side **Before-vs-After** data comparison panels.
    *   Sleek PowerShell-style real-time runtime log inspectors.
    *   Interactive data lineage and workflow structure visualizations.
    *   Secure, checksum-verified system installers with streamlined CSV export pipelines for financial/fraud mock datasets.
