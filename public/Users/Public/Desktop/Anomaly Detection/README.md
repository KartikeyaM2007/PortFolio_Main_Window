# Anomaly Detection Vision Transformer

### **Tech Stack**
`PyTorch` • `VideoMAE` • `OpenCV` • `PySide6` • `Python`

---

### **Key Features & Achievements**
*   **High Performance**: Built and trained a custom **9-Million parameter** VideoMAE + AnomalyTransformer model from scratch on **250GB+** of video data.
*   **Visual Feature Extraction**: Extracted robust **768-dimensional** visual features for high-accuracy defect detection.
*   **Unmatched Metrics**: Achieved a stellar **97.2% classification accuracy**, **0.98 AUROC**, and ultra-fast **<1s inference** response times.
*   **Interactive Desktop Application**: Packaged a full-featured Windows desktop GUI with support for:
    *   Uploaded offline video analysis
    *   Live webcam/camera anomaly detection feed
    *   Batch prediction on multiple streams
    *   Real-time anomaly timelines and score trend charting
    *   Detected frame grids and structured event summaries
