# Hiring Spree AI Agentic Resume Screening System

### **Tech Stack**
`Python` • `FastAPI` • `LlamaIndex` • `Gemini` • `React` • `Tailwind CSS`

---

### **Key Features & Achievements**
*   **Agentic Resume Screening**: Developed an advanced, automated recruitment application that leverage LLM agents to parser, evaluate, and rank resumes based on detailed job descriptions.
*   **Structured Parsing**: Utilized semantic retrieval and structural schemas extraction to safely extract candidate histories, degrees, skills, and experience metrics.
*   **Interactive Candidate Ranking**: Provided recruiters with a high-fidelity visual dashboard depicting candidate suitability scores, summary cards, and highlighted matching keywords.
*   **High Performance API**: Leveraged an asynchronous FastAPI backend for secure, non-blocking resume processing at scale.
