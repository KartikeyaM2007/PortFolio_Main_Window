# LeLo_InterView -- AI Interview Platform

### **Tech Stack**
`Next.js` • `TypeScript` • `Firebase` • `NextAuth` • `Vapi AI` • `Tailwind CSS`

---

### **Key Features & Achievements**
*   **AI-Powered Recruiting**: Engineered a full-stack automated interview platform that generates customized interview questions based on candidate resume uploads.
*   **Automated Evaluation**: Integrated robust ATS applicant tracking scoring models and structural AI feedback assessment panels.
*   **Vapi AI Integration**: Integrated realistic real-time conversational voice agents using `Vapi AI` for simulated phone/audio interview screening.
*   **Secure Infrastructure**: Added enterprise authentication via `NextAuth`, complete with protected router layers, real-time `Firebase` databases, and beautiful responsive interfaces styled using `Tailwind CSS`.
