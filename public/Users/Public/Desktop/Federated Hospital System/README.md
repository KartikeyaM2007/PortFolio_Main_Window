# Federated Hospital System

### **Tech Stack**
`TensorFlow.js` • `Django` • `Federated Learning` • `RBAC` • `Python` • `JavaScript`

---

### **Key Features & Achievements**
*   **Privacy-Preserving AI**: Developed a secure, cutting-edge federated learning platform allowing medical institutions to collaboratively train deep learning models without ever exposing sensitive patient records.
*   **Collaborative Machine Learning**: Trained **5 distinct models** on over **140,000+ data samples**, achieving a robust **92.69% validation accuracy**.
*   **Browser-Based Training**: Implemented client-side localized model training inside hospital staff web browsers using `TensorFlow.js`.
*   **Django Backend Integration**: Built secure, highly reliable aggregation APIs in Django, centralizing model weights while maintaining strict local data isolation.
*   **Enterprise Dashboard**: Added a comprehensive **Role-Based Access Control (RBAC)** dashboard with built-in client data validations (e.g. medical CSV schema checking) for administrators and researchers.
