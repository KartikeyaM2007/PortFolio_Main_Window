# GetFlowDone AI Agent Builder

### **Tech Stack**
`Next.js` • `React Flow` • `Convex DB` • `Gemini Flash` • `Clerk` • `Arcjet` • `Tailwind CSS` • `TypeScript`

---

### **Key Features & Achievements**
*   **Visual Workflow Builder**: Created an interactive node-based drag-and-drop workspace using `React Flow` with complete REST integration nodes, advanced conditional branch nodes, loop blocks, and modular tool-chaining.
*   **Advanced Data Flow**: Supported deep parameter extraction, robust input/output JSON schema validation, and instantaneous WebSocket-powered autosave syncing to `Convex DB`.
*   **AI Orchestration**: Integrated **Gemini Flash** LLM endpoints directly into active node triggers for real-time task reasoning and custom tools execution.
*   **Production-Grade Architecture**: Secured backend routes using **Arcjet** (rate limiting, bot detection) and authenticated sessions via **Clerk**.
*   **Robust Frontend Coverage**: Validated and built a production-optimized interface spanning **11 fully functional pages** with interactive workflow simulators, custom logs inspectors, and developer playgounds.
